### Summary

Next.js + Redis Database , db tool sample 

***
### required
* Next.js : 10.0.0
* Redis : 4.0.9
* node : 14.11
* react: 16.13.1

***
### Setup

npm install

***
### Setup , etc
* next.config.js , 

if change URL

```
BASE_URL: "http://localhost:3003",
```

* package.json / scripts

if change, port number ( -p )

```
"dev": "next dev -p 3003",
```

***
### start server
* Start :

yarn dev

* if change , release mode

yarn serve
